# Marlin Firmware – Ender‑3 Pro (Creality v4.2.7 + BLTouch)

This repository contains a community‑built Marlin firmware configuration
for the Creality Ender‑3 Pro using the Creality v4.2.7 board and BL‑Touch.

## Features
- Marlin 2.1.x
- Creality v4.2.7 (STM32)
- BL‑Touch support
- Bilinear mesh leveling
- Tuned defaults for reliability
